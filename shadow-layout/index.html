shadow-layout uses Shadow DOM, Data-binding and CSS flexbox to acheive many different layouts.

Examples:

    <shadow-layout id="myapp">
      <div id="header">1</div>
      <div id="info">2</div>
      <div id="toc">3</div>
      <div id="main">4</div>
      <div id="footer">5</div>
    </shadow-layout>
  
    myapp.layout =
      {flex: 1, flow: 'column', items: [
        {id: 'header'},
        {flex: 1, items: [
          {flow: 'column', items: [
            {id: 'info', flex: 1}, 
            {id: 'toc', flex: 2}
          ]}, 
          {flex: 1, flow: 'column', items: [
            {id: 'main', flex: 1},
            {id: 'footer'}
          ]}
        ]}
      ]};
  • Apply different layouts to the <shadow-layout> element

  • Adapt to different layout when screen size changes

  • Nested <shadow-layout>
