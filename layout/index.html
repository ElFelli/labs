g-layout uses Shadow DOM, MDV and CSS flexbox to acheive different layouts. It creates flex boxes inside shadow root and with Shadow DOM distribution allows for complex layout without needing to alter the DOM structure in the <g-layout> element.

Examples:

  • Apply different layouts to a <g-layout> element that contains 5 children
            <g-layout>
    
          <div class="box">1</div>
    
          <div class="box">2</div>
    
          <div class="box">3</div>
    
          <div class="box">4</div>
    
          <div class="box">5</div>
    
        </g-layout>
  • Adapt to different layout when screen size changes

  • Nested <g-layout>
